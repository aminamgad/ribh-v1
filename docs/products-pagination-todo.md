# قائمة مهام: أزرار السابق والتالي لصفحة المنتجات (الإدارة والمسوق)

## الهدف
إظهار أزرار **السابق** و **التالي** (وترقيم الصفحة "صفحة X من Y") بشكل واضح فوق أو تحت قائمة المنتجات، بحيث يمكن التنقل بين الصفحات ورؤية بقية المنتجات (مثلاً 20 لكل صفحة من أصل 365).

## النطاق
- **الإدارة (Admin):** جدول/شبكة المنتجات + أزرار الترقيم.
- **المسوق وتاجر الجملة (Marketer / Wholesaler):** شبكة أو جدول المنتجات + نفس أزرار الترقيم.
