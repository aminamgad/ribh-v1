# الفرق بين الرسائل والمحادثات (Messages vs Chat)

## ملخص سريع

| الجانب | الرسائل (Messages) | المحادثات (Chat) |
|--------|-------------------|------------------|
| المسار | /dashboard/messages | /dashboard/chat |
| الغرض | استفسارات المنتجات (مسوق/تاجر مع مورد) | محادثات الدعم (مستخدم مع إدارة) |
| المراجعة | تحتاج موافقة الإدارة | مباشرة بدون مراجعة |
| الربط بالمنتج | نعم | لا |

## الرسائل (Messages)
- النموذج: Message
- التدفق: إرسال رسالة لمورد بخصوص منتج، مراجعة الإدارة، اعتماد/رفض
- مراجعة الإدارة: /dashboard/admin/messages

## المحادثات (Chat)
- النموذج: Chat
- التدفق: إنشاء محادثة، تبادل رسائل مباشر
- المميزات: وقت فعلي، حالات نشط/مغلق، أولوية، تقييم

## التقييم: إبقاء النظامين
- الرسائل والمحادثات تخدم أغراضاً مختلفة ولا يُوصى بدمجهما حالياً.
- الرسائل: استفسارات المنتجات (مراجعة إدارية).
- المحادثات: دعم مباشر (بدون مراجعة).
